# thealxlabs.ca

Personal website and digital lab for [Alexander Wondwossen](https://thealxlabs.ca) — developer, creator, and photographer based in Toronto, CA.

## Pages

| Route | Description |
|---|---|
| `/` | Home — hero, terminal widget, now section |
| `/projects` | All 11 projects with tab filter + changelog |
| `/photos` | Architectural photography — Toronto grid series |
| `/about` | Bio, system metrics, tech stack, optics |
| `/uses` | Full gear list — hardware, software, camera, desk |
| `/notes` | Short-form lab notes on code, design, photography |
| `/stack` | Tech stack breakdown + skill bars |
| `/contact` | Links, availability, Toronto time |

## Stack

- **HTML / CSS / Vanilla JS** — no build step, no framework, ships fast
- **Fonts** — Bebas Neue + Space Mono via Google Fonts
- **Hosting** — Vercel
- **Domain** — thealxlabs.ca

## Deploy

This site is hosted on Vercel with clean URL routing via `vercel.json`. Push to `main` and it deploys automatically.

```bash
git clone https://github.com/thealxlabs/thealxlabs.ca.git
cd thealxlabs.ca
# edit files directly — no build step needed
```
